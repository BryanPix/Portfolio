Projets

Banana Organizer

Projet final concluant la formation de développeur web et web mobile pour le compte de l'Afpa de Saint-Étienne-du-Rouvray.

Création d'un gestionnaire de tâches sur le court et long terme, à l'aide d'une todolist et d'un calendrier.

Ce projet utilise le stack MERN :

  • MongoDB pour la base de données.
  • Express.js pour le framework backend.
  • React.js pour le framework frontend.
  • Node.js pour le serveur.
